The method

Built on one rule: never do the work for them.

A student who watches the answer appear feels like they learned — the fluency illusion — and remembers almost none of it. Socrates is designed around the opposite: productive struggle, then proof it stuck.

Socratic by design

It asks the question that makes you retrieve. If you stall, the question shrinks or a hint appears — the gap is never filled for you. For problem sets, it coaches your next step and never writes the solution.

Spacing that actually happens

Every session ends with the next reviews scheduled — +2, +5, +12, +30 days. Next time you show up, it greets you with what's due before anything new. If a test lands sooner, the schedule collapses into the runway.

Remembers each learner

One plain-markdown profile per student: level, subjects, what works, what's shaky, and the full review schedule. Profiles live outside the skill folder, so updates and reinstalls never touch them.

How a session works

Every session follows the same arc.

Light and conversational — it's a coach, not a form. Start however feels natural: paste notes, drop a link, photograph a problem, or just name a topic.

01

Load the learner

Returning students get greeted by name. First-timers answer four quick questions — under a minute — and a profile is saved.

02

Run due reviews first

Anything due or overdue gets a short retrieval hit — a five-item quiz, a brain dump, one past-paper question — before new material.

03

Take in what you brought

Notes, a reading, slides, a photo of the textbook, a link it actually reads, a problem set it refuses to solve for you — or just "I have a chem test on stoichiometry."

04

Diagnose, then pick a technique

Subject plus goal decide it — and if there's a test, the date shapes everything. It names the technique and why, so you learn the toolkit too.

05

Pick the mode

"Study with me" — it quizzes you live and keeps the rhythm. Or "coach me" — you drive, it sets up the reps and checks your output.

06

Run the session

Retrieval first, gaps worked, answers earned. Short focused blocks — and "I get it" gets verified with no-notes recall, not vibes.

07

Close the loop

The session is logged and the next reviews are scheduled, so the spacing actually happens. One honest line on whether it's sticking, and what to do next time.

The science

Re-reading. Highlighting. Cramming. Learning styles. None of it builds lasting memory. Retrieval and spacing do.

Most students don't have a study problem — they have a method problem. Time spent ≠ learning achieved. Almost every technique that works is a variation of two principles:

principle 01

Retrieval practice

Pulling information out of your head strengthens memory more than putting it in. Testing yourself badly beats reading perfectly.

principle 02

Spaced repetition

Reviewing just before you forget rebuilds the memory stronger. Sessions spread across days and weeks create retention that lasts past the test.

Readable by parents, by design

Profiles, logs, and review schedules are plain markdown files on your own machine — open them anytime and see exactly what's being studied and how it's going. No accounts, no app, no data leaving the computer, no internet needed at runtime. The full evidence base behind the coaching is published in the study guide.

Get started

Installed in under a minute.

The only requirement is Claude Code. The skill is pure markdown — no Python, no Node, no accounts — so it runs anywhere Claude Code runs.

Windows · PowerShell
git clone https://github.com/BourbonDog/Socrates.git
cd Socrates
Copy-Item -Recurse -Force ".\skills\socrates" "$env:USERPROFILE\.claude\skills\socrates"
macOS / Linux
git clone https://github.com/BourbonDog/Socrates.git
cd Socrates
mkdir -p ~/.claude/skills && cp -R skills/socrates ~/.claude/skills/socrates

Updating later? Pull and re-run the copy — profiles live outside the skill and are never touched. Then open Claude Code and start however feels natural:

Help me study for my biology test on cell respiration. I don't get this. (then paste notes or a photo) Quiz me on the French Revolution. Make me a study plan for finals.
